# Pitch Deck Conversion Guide

## Overview

The pitch deck is saved in Marp format: `pitch_deck_MarTech.md`

## Converting to Visual Slides

### Option 1: Marp (Recommended)
1. Install Marp CLI: `npm install -g @marp-team/marp-cli`
2. Convert to PDF: `marp pitch_deck_MarTech.md -o pitch_deck_MarTech.pdf`
3. Convert to HTML: `marp pitch_deck_MarTech.md -o pitch_deck_MarTech.html`
4. Convert to PowerPoint: `marp pitch_deck_MarTech.md -o pitch_deck_MarTech.pptx`

### Option 2: Online Marp Editor
1. Go to https://marp.app/
2. Copy content from `pitch_deck_MarTech.md`
3. Export as PDF, PowerPoint, or HTML

### Option 3: VS Code Extension
1. Install 'Marp for VS Code' extension
2. Open `pitch_deck_MarTech.md`
3. Use preview and export features

### Option 4: Manual Conversion
Copy each slide section to PowerPoint, Google Slides, or Keynote manually.
